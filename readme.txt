================================================
           FEMMECHIC — BOUTIQUE EN LIGNE
================================================

DESCRIPTION DU SITE
-------------------
FemmeChic est un mini site e-commerce dédié à la vente de
vêtements et accessoires pour femmes en Algérie.

Le site propose une sélection de robes, hauts, pantalons,
vestes et accessoires tendance, avec un design élégant et
une expérience utilisateur intuitive.

FONCTIONNALITÉS PRINCIPALES
----------------------------
- Page d'accueil avec hero, catégories et produits vedettes
- Boutique avec filtrage dynamique par catégorie et badge
- Recherche en temps réel et tri des produits
- Page produit avec modal de détail (taille, couleur)
- Panier latéral (cart drawer) avec gestion des quantités
- Wishlist (favoris)
- Inscription avec validation RegEx et jauge de sécurité
- Connexion avec session simulée via localStorage
- Page de commande en 3 étapes (livraison, paiement, confirmation)
- Codes promo (FEMME20, CHIC10, BIENVENUE)
- Design responsive (mobile, tablette, desktop)
- Menu de navigation sur toutes les pages

INSTRUCTIONS D'UTILISATION
----------------------------
1. Ouvrir index.html dans un navigateur moderne
2. Naviguer dans la boutique via le menu
3. Cliquer sur un produit pour voir ses détails
4. Ajouter au panier (bouton ou survol de la carte)
5. Cliquer sur l'icône panier pour voir le récapitulatif
6. Se connecter pour une expérience personnalisée
7. Passer commande depuis la page "Commande"
